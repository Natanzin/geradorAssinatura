# Gerador de Assinaturas de E-mail

Gerador de assinaturas institucionais em HTML desenvolvido para padronizar as assinaturas de e-mail dos colaboradores da **SUDECO**.

A aplicação permite preencher os dados do servidor, visualizar a assinatura em tempo real e exportá-la para utilização em clientes de e-mail como o Microsoft Outlook.

---

## ✨ Funcionalidades

* Geração de assinatura institucional em HTML
* Pré-visualização em tempo real
* Exportação da assinatura
* Layout responsivo
* Processamento totalmente local no navegador
* Não coleta nem armazena informações dos usuários

---

## 🛠️ Tecnologias utilizadas

* HTML5
* CSS3
* JavaScript (ES6)
* Bootstrap 5
* html2canvas

## 🚀 Como utilizar

1. Clone o repositório

```bash
git clone https://github.com/Natanzin/geradorAssinatura.git
```

2. Entre na pasta

```bash
cd geradorAssinatura
```

3. Abra o arquivo

```text
index.html
```

ou utilize qualquer servidor HTTP simples.

---

## 📋 Como funciona

1. Informe os dados do colaborador.
2. A assinatura é atualizada automaticamente.
3. Exporte a assinatura.
4. Importe o arquivo no cliente de e-mail.

---

## 🔒 Privacidade

Todo o processamento ocorre no navegador do usuário.

Nenhuma informação preenchida é enviada para servidores, armazenada em banco de dados ou compartilhada com terceiros.
